## TC-LOGIN-001 Successful login with valid credentials  
**Priority:** High  

### Preconditions    
- The user is on https://www.saucedemo.com/  
- The user is not logged in  

### Steps  
1. Type "standard_user" in the Username field  
2. Type "secret_sauce" in the Password field  
3. Click the *Login* button

- The user is directed to the inventory page (/inventory.html)  
- List of Products are visible
